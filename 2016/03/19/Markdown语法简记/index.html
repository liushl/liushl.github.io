Markdown语法简记

  • Markdown

刚刚改用Markdown，记录下基本语法，分享如下：

  1. 列表
    -或*无序列表
1.2.有序列表
    • 无序1
    • 无序2
  2. 引用
    > 加尖括号,例：>引用

    引用

  3. 图片
    ![](),例：![图片](http://7xrlxf.com1.z0.glb.clouddn.com/jianshu.jpg "头像")
  4. 链接
    []() ,例： [主页链接](http://eyow.xyz/)
    主页链接

  5. 粗体斜体

    两个*粗体，例：**粗体**

    粗体

    斜体一个*，例：*斜体*

    斜体

  6. 代码
    标记一小段行内两个｀把代码包含即可
    ("Hello word");
    代码块
    public void test(){
　System.out.println("Hello word");
}
  7. 分割线
    大于等于三个*、-都可以  例：---
    —
