Markdown语法简记

刚刚改用Markdown，记录下基本语法，分享如下：

  1. 列表
    -或*无序列表
1.2.有序列表
    • 无序1
    • 无序2
  2. 引用
    > 加尖括号,例：>引用

    引用

  3. 图片
    ![](),例：![图片](http://7xrlxf.com1.z0.glb.clouddn.com/jianshu.jpg "头像")
  4. 链接
    []() ,例： [主页链接](http://eyow.xyz/)
    主页链接

  5. 粗体斜体

    两个*粗体，例：**粗体**

    粗体

    斜体一个*，例：*斜体*

    斜体

  6. 代码
    标记一小段行内两个｀把代码包含即可
    ("Hello word");
    代码块
    public void test(){
　System.out.println("Hello word");
}
  7. 分割线
    大于等于三个*、-都可以  例：---
    —

常用Android Studio插件

  1. android-butterknife-zeleznyandroid-butterknife-zelezny
    视图绑定，能够方便快速初始化
  2. Android Parcelable code generator
    帮助实体bean生成Parcelable接口
  3. Gsonformat
    根据json数据快速生成java实体类
  4. Android Drawable Importer
  5. Lifecycle Sorter
    根据Activity或者fragment的生命周期对其生命周期方法位置进行先后排序
  6. JsonOnlineViewer
    直接在android studio中调试接口数据
  7. Android Code Generator Plugin
    自动生成findViewById ，类似插件蛮多，有了butterknife，这个就不怎么用
